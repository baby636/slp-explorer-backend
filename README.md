## SLP explorer backend

[Slp explorer](https://github.com/nestjs/nest) Explore Bitcoin Cash tokens with analytics and transactions.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
